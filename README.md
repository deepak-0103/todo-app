# To-Do List Application

A simple and interactive To-Do List application built using **React (Vite)**.  
This application allows users to add, edit, delete, and mark tasks as completed.

----

##  Features

-  Add new tasks  
-  Mark tasks as completed on clicking on the task 
-  Delete tasks  
-  Edit existing tasks  
-  Clean and user-friendly UI  
-  Built using Vite for fast development  

----

##  Technologies Used

- React (Functional Components)
- Vite
- JavaScript (ES6)
- CSS

---

STEPS ## ▶ How to Run the To-Do Application

Follow the steps below to run the project on your local machine.

### 1️⃣ Clone the Repository

git clone https://github.com/YOUR_USERNAME/todo-app.git

2️⃣ Navigate to the Project Folder
cd todo-app
3️⃣ Install Dependencies

Make sure Node.js is installed on your system.

npm install

This will install all required packages listed in package.json.

4️⃣ Start the Development Server
npm run dev

After running this command, you will see output similar to:

Local: http://localhost:5173/
5️⃣ Open in Browser

Open your browser and go to:

http://localhost:5173/

You should now see the To-Do List application running.
